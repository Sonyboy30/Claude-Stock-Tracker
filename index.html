How to Add a New Stock GitHub Pages Guide
📋 4-Step Process to Add Any Stock
01
Generate the Report
Ask Claude to build a full risk report for your new ticker (e.g. "Build a risk report for NVDA like the MSFT one"). Use the same dark design system.
02
Upload to GitHub
Go to your repo → click "Add file" → "Create new file". Name it TICKER.html (e.g. NVDA.html). Paste the HTML and commit.
03
Update This Homepage
Edit index.html — copy an existing stock card, update the ticker/price/data, and change the href to "TICKER.html". Add a row to the table too.
04
Done — It's Live!
GitHub Pages auto-deploys in ~1 min. Your new stock card appears on the homepage and links directly to the report. No build step needed.
