➕ Add a New Stock Report

Follow these steps to add any stock to your tracker in under 5 minutes.

1
Ask Claude — Open a new Claude chat and say:
"Build a dark HTML risk report for [TICKER] matching the Stock Risk Tracker design at sonyboy30.github.io/Claude-Stock-Tracker"
2
Save the file — In your GitHub repo, click "Add file" → "Create new file" → name it TICKER.html → paste Claude's output → Commit.
3
Add to index.html — Edit this file, find the STOCKS array near the bottom, copy one entry and update the values for your new ticker.
4
Add to sectors.html — Find the matching sector in SECTORS array and add the stock there too. Both pages auto-update from data.
